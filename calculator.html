Graphing Calculator (Java)

source code
  • November 2015 - March 2016
  • approx. 75+ hours of research and work

This was my diploma project in my final year of high school (more information on diploma projects in Swedish high schools available here). Two of my classmates and I set out to create a graphing calculator based off a Raspberry Pi, and I was made solely responsible for creating the software while they worked on the casing.

The entirety of the code was written in Java, with the help of the NetBeans IDE to create the necessary menu structure. In the end the code ended up both messy and inefficient (and, honestly speaking, ridden with bugs), but I was nonetheless proud to have completed the task as this was not only my first large scale programming project, but also the first time I had created a piece of software from the ground up.

The project taught me a lot about programming, both in Java specifically and in general. I was very much a novice when we started out so I had to learn a lot of the concepts and techniques as I was going, essentially meaning even the smallest of tasks took a lot of brain-storming and effort to figure out. It was both a very demanding and fun process!

Finished Product

Though we ultimately failed to produce a final product as we did not have the means to create the casing, the software ran successfully on the Raspberry Pi. It wasn't neccessarily very pretty ( *cough* ) but interacting with it using the touchscreen we connected was as effortless as we had hoped for.
